### About

**FeatherAPI** is a simple and lightweight REST API testing tool, inspired by Postman, but designed to consume minimal system resources.

It was built using **Java Swing** with a focus on simplicity and practicality.

### Features

- Support for HTTP methods:
  - GET
  - POST
  - PUT
  - DELETE
  - PATCH
- Minimalist and fast UI
- Low memory and CPU usage
- Easy to use, no bloated features

### Running Locally

Make sure you have **Java** and **Maven** installed.

Run the following command:

```bash
mvn clean compile exec:java
```
